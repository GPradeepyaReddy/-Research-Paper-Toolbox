How to Organize and Analyze Research Data

Back to Home

For this lesson, you'll need:

  • Microsoft Excel
  • PASW statistics software (or free trial)

Prerequisites:

  • Basic Excel functionality
  • Descriptive & inferential statistics
  • Understanding IVs and DVs

Lesson Outcomes

  • Format research data in Excel
  • Run descriptive statistics in Excel
  • Prepare data for PASW

Format Data for Analysis

Open ToolBox_DataLesson.xls. In the Raw_Dataset tab:

  • Ensure all variable names are unique
  • Fix invalid variable names (e.g. 4tutorial_types → tutorial_types)
  • Rename columns for clarity

Format data entries consistently, and split compound fields into separate columns (e.g. tasks_completed → task1, task2, task3). Use 0/1 for dichotomous values.

Dichotomous variables enable isolation for comparative analysis in PASW.

Copy data to Sheet2, rename it Formatted_Data, and add new columns per categorical value.

Run Descriptive Statistics

Example insight: perc_enjoy > perc_usefulness with high variance → possible disengaged segment.

  1. Highlight continuous values (e.g., perc_enjoy to perc_ease_of_use)
  2. Tools > Add-Ins → Enable Analysis ToolPak
  3. Tools > Data Analysis > Descriptive Statistics → Select range → Output to new sheet

Run Inferential Statistics

  1. Open PASW → Load Excel file → Select Formatted_Data
  2. Analyze menu → Explore Wilcoxon-Mann-Whitney, Linear Regression, Correlation
  3. Use Kruskal-Wallis test: Analyze > Nonparametric Tests > K Independent Samples

Export results to Word, PDF, etc. File > Export → Choose format

Use UCLA's "Choosing the Correct Statistic" to guide statistical choices.
